# Simple Viewer application : aps-example
##  How to run:
- Open the application using vscode and select Run > Start Debugging from upper left menu
- Before that a .env file needs to be configured. In the project directory create a .env file wich should contain the following VARIABLE_NAMES:
    
        1. APS_CLIENT_ID
        2. APS_CLIENT_SECRET
        3. APS_BUCKET
        4. PORT
    Create a .env file
    ```
    > cd aps-example && touch .env
